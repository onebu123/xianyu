# v2.0.0 验收清单

## 发布门禁

- [x] 根项目、后端、前端版本号统一为 `2.0.0`。
- [x] 旧发布入口已移除。
- [x] `npm run release:v2` 作为唯一正式发布入口。
- [x] 发布包包含 v2 交付文档和发布清单。

## 私有化模式

- [x] `APP_DEPLOYMENT_MODE=private` 可启动。
- [x] 单步登录兼容旧私有化链路。
- [x] 本地演示 SQLite 兼容。
- [x] 企业正式私有化支持 PostgreSQL 基线。

## SaaS 模式

- [x] `APP_DEPLOYMENT_MODE=saas` 可启动。
- [x] 控制面支持租户创建、暂停、恢复和查询。
- [x] 控制面支持租户成员管理和初始管理员分配。
- [x] 登录后返回可访问租户列表。
- [x] 选择租户后签发租户作用域会话。
- [x] 平台接口拒绝租户 token。
- [x] 业务接口拒绝平台 token。

## 数据隔离

- [x] 租户业务库通过 resolver 获取，不由客户端传入。
- [x] 新租户可初始化独立业务库。
- [x] 跨租户读写被拒绝。
- [x] 控制面不保存租户业务数据。

## 队列与 Worker

- [x] API 进程只负责入队。
- [x] Worker 独立运行。
- [x] 任务包含 `tenantId`、`jobId`、幂等键、重试策略和死信状态。
- [x] Redis 队列冒烟通过。
- [x] Worker 干运行通过。

## 前端与交付

- [x] 登录、选租户、进入看板链路可冒烟。
- [x] 私有化 Web 冒烟通过。
- [x] SaaS Web 冒烟通过。
- [x] CI 可按 v2 矩阵拆分执行。
