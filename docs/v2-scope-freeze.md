# v2.0.0 范围冻结说明

## 冻结范围

`v2.0.0` 冻结为企业级 SaaS + 私有化双模式基线，包含控制面、多租户认证、独立租户业务库、Redis 队列、Worker、多店铺运营、开放平台、安全治理、可观测性和交付治理。

## 不进入本版本

- 企业自助注册、商业计费、渠道结算。
- SSO、OIDC、LDAP、SAML。
- SQLite 到 SaaS 多租户在线迁移。
- 多租户共享库方案。
- 绕过平台规则的自动化能力。
