# 对标闲鱼商家平台路线图

本项目的目标是建设类似企业级闲鱼商家运营平台的后台能力，但所有接入必须遵守平台规则、账号授权和客户合规边界。

## 当前版本

| 版本 | 名称 | 状态 |
| --- | --- | --- |
| `v2.0.0` | 企业级 SaaS + 私有化正式基线 | [x] |
| `v2.1` | 真实店铺小流量联调版 | [ ] |
| `v2.2` | 商业化与自助开通版 | [ ] |
| `v2.3` | 企业身份与高级权限版 | [ ] |
| `v2.4` | 生态连接器版 | [ ] |

## v2.0.0 已完成

- [x] SaaS 控制面。
- [x] 私有化兼容。
- [x] 每租户独立业务库。
- [x] PostgreSQL 正式基线。
- [x] Redis 队列和 Worker。
- [x] 多店铺运营中心。
- [x] 订单、售后、资金、履约、AI 工作台。
- [x] 开放平台、Webhook、API Key。
- [x] MFA、密钥治理、审计和可观测性。
- [x] 发布、验收、部署、回滚和交付文档。

## v2.1 重点

- 接入真实测试店铺。
- 建立同步频率、失败退避、登录态续期和人工介入机制。
- 建立真实订单、商品、售后、消息链路的灰度验收。
- 输出真实生产运行周报和容量基线。

## v2.2 重点

- 企业自助注册。
- 套餐、用量、计费和账单。
- 渠道客户和客户成功运营看板。
- 自动化租户开通流程。

## v2.3 重点

- SSO、OIDC、LDAP、SAML。
- 更细粒度 RBAC、审批流和操作保护。
- 外部 KMS 或 Secret Manager。

## v2.4 重点

- ERP、WMS、财务系统、客服系统连接器。
- 更完整的开放平台开发者门户。
- Webhook 订阅治理、签名验签和重放保护。
