# 生产化推进路线

## 目标

`v2.0.0` 已作为企业级 SaaS + 私有化双模式基线，后续生产化工作的重点是围绕真实客户环境做容量、合规、告警和值班闭环。

## 已完成

- [x] 私有化和 SaaS 运行模式。
- [x] 控制面、租户模型、平台账号和租户会话。
- [x] PostgreSQL 正式数据库基线。
- [x] Redis 队列与独立 Worker。
- [x] 结构化日志、请求 ID、健康检查和 Prometheus 指标。
- [x] 数据库巡检、备份恢复、回滚和事故响应文档。
- [x] v2 发布打包和验收清单。

## 下一步

- [ ] 基于真实客户店铺做小流量试运行。
- [ ] 建立生产告警接入样例。
- [ ] 建立压测脚本和容量基线报告。
- [ ] 建立值班、故障复盘和升级演练制度。
- [ ] 接入外部 KMS、SSO 和商业计费系统。

## 生产准入

- 完成 `npm run release:v2`。
- 完成真实店铺授权与合规确认。
- 完成备份恢复演练。
- 完成告警接入和责任人配置。
- 完成灰度发布和回滚演练。
