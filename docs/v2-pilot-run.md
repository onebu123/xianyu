# v2.0.0 试运行记录

## 试运行范围

- 私有化启动、构建、发布打包。
- SaaS 控制面租户创建、选租户登录、成员关系和隔离校验。
- PostgreSQL 控制面和租户业务库初始化。
- Redis 队列、Worker 干运行和任务状态闭环。
- 前端私有化和 SaaS 登录看板冒烟。

## 结论

- v2 发布链路以 `npm run release:v2` 为准。
- v2 文档、版本号、打包脚本和交付清单已统一。
- 真实闲鱼生产接入前仍需要客户提供合法账号、授权店铺、风控边界和生产环境配置。

## 后续试运行建议

- 先使用一个真实测试店铺做低频同步。
- 再扩大到多店铺、多成员、多任务并发。
- 最后接入告警、备份恢复演练和正式运维值班。
